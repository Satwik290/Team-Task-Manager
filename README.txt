[OVERVIEW]
Team Task Manager is a high-performance, Tier-1 scalable management ecosystem.
It delivers a visually immersive 'Glassmorphism' surface combined with 
industrial-grade defensive backend engineering.

[CORE ARCHITECTURE]
- ENGINE: Next.js 15 (App Router) with React 19 concurrent features.
- DATA: PostgreSQL + Prisma ORM with atomic transaction safety.
- AUTH: Hardened JWT-based secure session management (bcrypt + jose).
- STYLE: Vanilla Tailwind CSS with custom Glassmorphism primitives.
- OPS: Turborepo-managed monorepo for maximum build velocity.

[ELITE FEATURES]
- OPTIMISTIC UI: Zero-latency task status transitions.
- DYNAMIC WORKSPACE: Seamless toggle between Kanban Board and Calendar views.
- ROLE-BASED ACCESS: Strict RBAC gating for Admin and Member workflows.
- FULL-CRUD MODAL: State-of-the-art task orchestration interface.

[QUICK START PROTOCOL]
1. PREPARE: Copy .env.template to .env and configure JWT_SECRET/DATABASE_URL.
2. INSTALL: Run `npm install` to provision dependencies.
3. MIGRATE: Run `npm run db:generate && npm run db:push` to sync the schema.
4. LAUNCH:  Run `npm run dev` to start the local engine.

[ACCESS CONTROL]
- ADMIN: Complete system oversight, project creation, and teammate management.
- MEMBER: Streamlined workspace for viewing and updating assigned work.
